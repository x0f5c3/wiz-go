## Installation

Run the following command in a terminal and you're ready to go!

**Windows**
```powershell
iwr instl.sh/x0f5c3/wiz-go/windows | iex
```

**macOS**
```bash
curl -sSL instl.sh/x0f5c3/wiz-go/macos | bash
```

**Linux**
```bash
curl -sSL instl.sh/x0f5c3/wiz-go/linux | bash
```
